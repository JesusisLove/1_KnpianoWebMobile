## 四、各视图判断理由

### v_info_lesson_fee_connect_lsn　→　废弃视图

系统中没有任何代码引用此视图。它是旧版课费与课程关联视图，已被功能更完善的新版视图 `v_info_lesson_fee_connect_lsn_and_extraToScheDataCorrect` 完全取代。两者的主要差异如下：

| 比较项目 | v_info_lesson_fee_connect_lsn（旧） | v_info_lesson_fee_connect_lsn_and_extraToScheDataCorrect（新） |
|---------|-----------------------------------|------------------------------------------------------------|
| bad_debt_flg | 无 | 有（支持坏账过滤） |
| JOIN方式 | LEFT JOIN（v_info_student_document） | INNER JOIN（v_info_student_document） |
| 临时课程支持 | 无 | 有（UNION ALL 临时课部分） |
| 数据源视图 | v_info_lesson_fee_include_extra2sche | v_info_lesson_fee_and_extraToScheDataCorrect |

### v_info_lesson_fee_include_extra2sche　→　连带废弃

唯一引用此视图的地方是 `v_info_lesson_fee_connect_lsn`（上述废弃视图）的定义内部。由于 `v_info_lesson_fee_connect_lsn` 本身是废弃视图，此视图实质上也无人使用，可以连带删除。

### v_info_lesson_include_extra2sche　→　现役视图，需保留

此视图被 Web Mapper XML 共5处实际引用，是现役运行中的视图，不可删除。需要在 `Views/` 文件夹下补充创建对应的单独文件，以保持与其他视图的管理一致性。
